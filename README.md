## Getting Started

Clone the repository

`git clone https://github.com/barend-erasmus/redis-manager-service.git`

Change to cloned directory

`cd ./redis-manager-service`

Install node packages

`npm install`

Start project

`npm start`

Browse `http://localhost:3000/api`
